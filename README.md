# 🌿 BloomSphere – Smart Plant Store

BloomSphere is a modern plant e-commerce website built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add to cart with localStorage
- Quantity management
- Search functionality
- Responsive design
- Checkout page

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 📂 Project Structure
- css/ → Stylesheets
- js/ → JavaScript logic
- images/ → All product images
