# 🌿 BloomSphere – Smart Plant E-Commerce Store

BloomSphere is a modern, responsive plant e-commerce website built using HTML, CSS, and Vanilla JavaScript.  
It simulates a real online shopping experience with cart management and checkout functionality using browser storage.

---

## 🚀 Features

- 🛒 Add to Cart with LocalStorage persistence
- 🔢 Quantity management system
- 🗑 Remove items from cart
- 💰 Automatic total price calculation
- 🔍 Live search functionality
- 💳 Demo checkout & payment page
- 📱 Fully responsive design
- ✨ Smooth UI animations & modern layout


---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Styling, layout & responsiveness
- **JavaScript (Vanilla JS)** – Dynamic functionality
- **LocalStorage API** – Client-side cart persistence
